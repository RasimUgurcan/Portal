# SEO Optimizasyon Rehberi - EYS Global

## Yapılan SEO İyileştirmeleri

### 1. Meta Tags ✅
- Title tag optimize edildi
- Description tag genişletildi ve anahtar kelimeler eklendi
- Keywords meta tag eklendi
- Author, robots, language meta tag'leri eklendi

### 2. Open Graph Tags ✅
- Facebook/LinkedIn paylaşımları için Open Graph tag'leri eklendi
- Twitter Card tag'leri eklendi
- Sosyal medya paylaşımlarında görsel ve açıklama görünecek

### 3. Structured Data (Schema.org) ✅
- Organization schema eklendi
- WebSite schema eklendi
- Google'ın sitenizi daha iyi anlaması için JSON-LD formatında veri eklendi

### 4. robots.txt ✅
- Arama motorları için robots.txt dosyası oluşturuldu
- Portal ve API alanları engellendi
- Önemli sayfalar allow edildi

### 5. sitemap.xml ✅
- Tüm sayfalar için XML sitemap oluşturuldu
- Priority ve changefreq değerleri ayarlandı
- Google Search Console'a eklenebilir

### 6. Alt Text İyileştirmeleri ✅
- Tüm görseller için açıklayıcı alt text'ler eklendi
- SEO için önemli anahtar kelimeler alt text'lere eklendi

### 7. Canonical URL ✅
- Canonical URL tag'i eklendi
- Duplicate content sorununu önler

## Yapılması Gerekenler

### Hemen Yapılacaklar:

1. **Google Search Console'a Kayıt**
   - https://search.google.com/search-console adresine gidin
   - Sitenizi doğrulayın (HTML tag veya DNS ile)
   - sitemap.xml'i gönderin: https://eysglobal.com.tr/sitemap.xml

2. **Google Analytics Kurulumu**
   - Google Analytics hesabı oluşturun
   - Tracking code'u tüm sayfalara ekleyin
   - Conversion tracking ayarlayın

3. **Google My Business**
   - İşletme bilgilerinizi Google My Business'a ekleyin
   - Adres, telefon, çalışma saatleri bilgilerini güncelleyin
   - Müşteri yorumlarını yönetin

4. **Sosyal Medya Hesapları**
   - LinkedIn, Facebook, Instagram hesapları oluşturun
   - Her sayfada sosyal medya linklerini ekleyin
   - Düzenli içerik paylaşın

### İçerik Optimizasyonu:

1. **Blog/İçerik Bölümü Ekleyin**
   - SEO için blog yazıları yazın
   - "ISO sertifikasyonu nedir?", "Sosyal uygunluk denetimi nasıl yapılır?" gibi konular
   - Düzenli içerik güncellemeleri yapın

2. **Sayfa İçi SEO**
   - Her sayfada H1, H2, H3 başlıklarını doğru kullanın
   - İç linkleme yapın (örneğin: "ISO sertifikasyonu" yazısına link ekleyin)
   - Anahtar kelimeleri doğal şekilde kullanın

3. **Sayfa Hızı Optimizasyonu**
   - Görselleri optimize edin (WebP formatı kullanın)
   - CSS ve JS dosyalarını minify edin
   - CDN kullanmayı düşünün

4. **Mobil Uyumluluk**
   - Responsive tasarım zaten var ✅
   - Google Mobile-Friendly Test ile kontrol edin

### Teknik SEO:

1. **HTTPS**
   - SSL sertifikası kurulu olmalı ✅
   - Tüm sayfalar HTTPS üzerinden servis edilmeli

2. **Sayfa Hızı**
   - Google PageSpeed Insights ile test edin
   - Gerekirse görsel optimizasyonu yapın
   - Lazy loading ekleyin

3. **404 Sayfaları**
   - Özel 404 sayfası oluşturun
   - Kullanıcıları ana sayfaya yönlendirin

### Backlink Stratejisi:

1. **Yerel Dizinler**
   - Türkiye'deki işletme dizinlerine kayıt olun
   - Sektörel dizinlere ekleyin

2. **İçerik Pazarlama**
   - Değerli içerikler üretin
   - Diğer sitelerin sizi referans göstermesini sağlayın

3. **Sosyal Sinyaller**
   - LinkedIn'de şirket sayfası oluşturun
   - Düzenli paylaşımlar yapın

### Ölçümleme ve Takip:

1. **Google Analytics**
   - Hangi sayfaların en çok ziyaret edildiğini takip edin
   - Kullanıcı davranışlarını analiz edin

2. **Google Search Console**
   - Hangi kelimelerle bulunduğunuzu görün
   - Tıklama oranlarını takip edin
   - Hataları düzeltin

3. **Keyword Tracking**
   - Anahtar kelimelerinizi takip edin
   - Sıralamalarınızı izleyin

## Sonuç

Temel SEO optimizasyonları tamamlandı. Şimdi:
1. Google Search Console'a kayıt olun
2. sitemap.xml'i gönderin
3. Düzenli içerik güncellemeleri yapın
4. Backlink stratejisi uygulayın
5. Performansı sürekli takip edin

Bu adımları takip ederek Google'da ilk sıralarda çıkma şansınızı artırabilirsiniz.
